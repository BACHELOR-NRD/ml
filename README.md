# ML CarBucks

ML CarBucks is a bachelor project developed in collaboration with a company in 2025. The project focuses on building a machine learning pipeline using computer vision models to detect car damages. The goal is to automate the identification of issues in vehicles, making the process faster and more reliable.

## Features

- Computer vision models for damage detection
- Automated analysis of car images
- Streamlined workflow for identifying car issues

## Purpose

This project aims to assist companies in quickly and accurately assessing vehicle conditions, reducing manual inspection time and improving efficiency.

## Status

Currently under active development as part of a bachelor thesis.
